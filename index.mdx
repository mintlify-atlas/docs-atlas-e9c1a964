CS2 SDK (`cs2-sdk.exe`) is an all-in-one external dumper for Counter-Strike 2. Run it once while CS2 is open and it produces a dated session folder containing every offset, schema class, interface RVA, button hash, and signature your cheat or research project needs — in every language you might use.

## What CS2 SDK produces

A single run performs two passes against the live CS2 process and writes everything into one dated folder:

<CardGroup cols={2}>
  <Card title="Offsets pass" icon="database">
    Schema classes, interfaces, buttons, and raw module offsets in `cs`, `hpp`, `json`, `rs`, and `zig` formats — plus a typed C++ SDK amalgamation.
  </Card>
  <Card title="Signatures pass" icon="magnifying-glass">
    Every configured IDA-style pattern resolved to an RVA, with automatic diff against the previous session.
  </Card>
  <Card title="Typed SDK headers" icon="brackets-curly">
    `cs2sdk.hpp` — a single-include C++ header with `SCHEMA_FIELD` typed accessors, resolved interfaces, netvars, and signature constants.
  </Card>
  <Card title="Vtable dump" icon="table">
    Per-interface vtable indices with MSVC RTTI class-name recovery and method names sourced from the signature database.
  </Card>
</CardGroup>

## Get started

<Steps>
  <Step title="Build the binary">
    Clone the repo and run `cargo build --release`. Requires Rust 1.85+ and Windows 10/11.
  </Step>
  <Step title="Launch CS2">
    Start Counter-Strike 2 and leave it running at the main menu or in-game.
  </Step>
  <Step title="Run as Administrator">
    Execute `.\target\release\cs2-sdk.exe` in an elevated terminal. The tool attaches, dumps, and exits cleanly.
  </Step>
  <Step title="Use the output">
    Find your output under the `output\` folder (or whatever path you passed to `--output`). The most useful single file for C++ consumers is `sdk\cs2sdk.hpp`.
  </Step>
</Steps>

<Note>
  CS2 SDK requires Administrator privileges when using the default `memflow-native` backend. The tool exits cleanly if CS2 is not running.
</Note>

## Language support

CS2 SDK emits offsets and signatures in multiple languages:

| Language | Offsets / schemas | Signatures | Typed SDK |
|----------|:-----------------:|:----------:|:---------:|
| C++ | `buttons.hpp` | `signatures.hpp` | `cs2sdk.hpp`, `sdk/<module>.hpp` |
| C# | `buttons.cs` | `signatures.cs` | — |
| Rust | `buttons.rs` | `signatures.rs` | `cs2sdk.rs` |
| Zig | `buttons.zig` | — | — |
| JSON | `buttons.json`, `netvars.json` | `signatures.json` | `vtables.json` |
